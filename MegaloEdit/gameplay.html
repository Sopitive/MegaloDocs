Gameplay Configuration

Learn how to define teams, loadouts, and general game options for your gametype.

Teams

You can define up to eight teams, including their model (Spartan/Elite), names, designators, and colors.

teams
    model spartan ; Set the player species of all teams  
    team ; Index 0
        name "Defenders"
        designator defenders
        fireteam_count 2 
        color 255 0 0 ; Red
        model elite ; Override specific team model
    end
end

Setting the designator to none will make everyone on that team an enemy to everyone else (even each other).

Loadouts

MegaloEdit allows you to create custom loadouts and loadout palettes. Note that the number of referenceable loadouts is limited (typically to 32).

loadout my_loadout
    name loadout_name_noble1 ; Predefined options
    primary_weapon spike_rifle
    grenades 4 each ; Frag, plasma, each
end

loadout_palette my_palette
    item my_loadout
end

Game Options

Game options define how the match is played, from score limits and round times to respawn penalties.

game_options
    override score_to_win_round 0 ; Unlimited
    override fire_teams_enabled true
    override respawn_time 5 ; Total respawn time in seconds
    
    ; Custom options for players
    option custom_option
        "Game Speed"
        "Sets the movement speed bonus."
        0 ; Default
        0 "Normal" ""
        1 "Double" ""
    end
end

Using lock or hide before an override prevents players from changing it in the pre-game menu.

Player Traits

Player traits modify the physical attributes of players, such as their movement speed, gravity, and weapon modifiers.

override base_player_traits
    damage_resistance invulnerable
    speed 150
    gravity 140
    infinite_ammo true
end

; Custom traits to be applied later in scripts
player_traits custom_trait
    "Berserker"
    "Increased speed but vulnerable."
    speed 200
    damage_resistance 10
end
