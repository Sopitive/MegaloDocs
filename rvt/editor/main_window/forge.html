Forge Settings

The Forge settings page allows you to edit options related to Forge. It only appears when opening a Forge variant file.

The fields available to edit are as follows:

Enable Open-Channel Voice
Unknown.
Unknown Flag 1
Unknown.
Editor Respawn Time
Presumably controls how long it takes for players to respawn if they die while in Edit Mode; however, testing suggests it may not be used.
Edit Mode Access
Unknown.
