Reach Variant Tool

Reach Variant Tool is a tool created by DavidJCobb for creating, modifying and compiling gametypes for Halo: Reach. This tool uses a custom scripting language based on Lua to create and compile gametypes.

Ksoft

Ksoft is a tool created by Kornman00 used for creating, editing, and compiling gametypes for Halo: Reach. Unlike some gametyping methods, Ksoft uses XML files in the creation of gametypes.

Megalograph

Megalograph is a tool created by Gamergotten used for creating, editing, and compiling gametypes in Halo 2: Anniversary. Gametypes are created via a node graph instead of a written script.

Downloads:

Github

Halo: Reach Mod Tools

The Halo: Reach Mod Tools are the official mod tools released by 343 Industries. They contain tools for creating modded content within Halo: Reach. Included in these tools is MegaloEdit, which is used to create, edit, and compile gametypes in Halo: Reach. This tool uses the official Megalo scripting language.

Downloads:

Steam

Mjolnir Forge Editor

Mjolnir Forge Editor is a plugin for Blender that is used to create custom map variants within Halo: Reach.
