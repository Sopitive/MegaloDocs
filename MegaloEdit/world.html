World & Forge Interaction

Learn how to define map objects and Forge labels to interact with the physical world in your gametype.

Map Objects & Labels

Map objects are the physical items on a map that your script can reference. You can filter all objects on a map based on their Forge label, team, type, or spawn sequence.

map_object referenceable_name
    label "my_label" ; Name seen in Forge
    min 1 ; Minimum number of objects with this label required to play
end

map_object health_kits
    type "health_pack_equipment" ; Reference all health kits
end

Forge Labels

Defining labels here allows Forgers to see them in the Forge menu and assign them to objects. Use min to require a map to have a minimum number of objects with that label.

Triggers for Map Objects

You can create specialized triggers that loop through all objects with a specific Forge label.

trigger referenceable_name
    condition if current_object != none
    action delete_object current_object
end
