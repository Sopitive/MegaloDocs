Player Rating Parameters

The player rating parameters script allows you to tune how the game variant affects a player's Arena rating. Arena was the system that Bungie designed for Halo: Reach in an attempt to rank players by skill. It is no longer used; 343 Industries relies on TrueSkill 2, a system that uses kills per minute, deaths per minute, and machine learning to determine a player's skill level.

The fields available to edit are as follows:
