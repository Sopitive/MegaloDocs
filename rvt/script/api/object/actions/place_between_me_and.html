object.place_between_me_and

Spawn a new object between this object and another.

This function returns object.

Arguments

other

The other object to use as a basis.

type

A type of object that can exist in the game world.

radius

The radius the object is allowed to spawn near the target object.

Example


global.object[0] = global.object[1].place_between_me_and(global.object[2], skull, 0)

Notes

  • Unlike object.place_at_me, this function appears to spawn objects exactly in place, provided they aren't solid or obstructed. It will also place an object "looking" straight upward.
