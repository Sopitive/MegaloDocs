HUD Widget

A script-defined HUD widget.

Member actions

set_icon

Set the HUD widget's icon.

set_meter_params

Set the HUD widget's meter parameters.

set_text

Set the HUD widget's text.

set_value_text

Set the HUD widget's value text.

set_visibility

Control whether a player can see this HUD widget.
