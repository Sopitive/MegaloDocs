Scripted Stats

The scripted stats page allows you to define custom scoreboard stats that can be used by the gametype script. Note that as of this writing, Halo: The Master Chief Collection does not display these stats anywhere.

The fields available to edit for each set of player traits are as follows:

Name
The string used for the traits name.
Format
Controls how the stat's value is displayed on the scoreboard.
Sort Order
Determines whether and how players are sorted within the scoreboard by this stat's value.
Group Players by Team
Unknown.

Scripted stats can be used to show gametype-specific statistics on the scoreboard, such as the number of times a player has armed and detonated a bomb in Assault. They are also a useful means of persisting data across rounds, so that a script can know what happened in the previous round(s).
