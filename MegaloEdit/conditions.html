MegaloEdit Conditions

Conditions are used in triggers to control whether actions should be executed. A trigger will only process its content if all specified conditions evaluate to true.

General Conditions

if

The basic comparison condition used for variables and numbers.

Example: condition if global_number == 5

not

Inverts the result of the following condition.

Object Conditions

object_is_type

Checks if the specified object matches a particular object type (e.g., "monitor", "warthog").

RVT Equivalent: object.is_of_type

object_matches_filter

Checks if an object matches a defined map_object filter (by label, sequence, etc).

Player Conditions

player_is_spartan

Checks if the player's current species is a Spartan.

RVT Equivalent: player.is_spartan

player_is_elite

Checks if the player's current species is an Elite.

RVT Equivalent: player.is_elite
