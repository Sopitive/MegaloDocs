Player Traits

Traits can be applied to each player using script_traits[n], where n is equal to the index of the traits to be applied.

Example

for each player do
    if current_player.biped.is_of_type(spartan) then
        current_player.apply_traits(script_traits[0])
    end
end

Notes

  • Traits are only effective for the tick in which you applied them. If you wish to keep the traits active for longer than that, you must apply the traits each game tick.

  • Applying traits on top of other traits will cause the second traits to take precedance over the first.
