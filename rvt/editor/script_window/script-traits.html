Scripted Traits

The scripted traits page allows you to define player trait sets for the gametype, which can be viewed and configured in-game. You can add and reorder traits, and you can delete any traits that are not actively in use by the gametype script.

The fields available to edit for each set of player traits are as follows:

Name
The string used for the traits name.
Description
The string used for the traits description.

To edit the actual player traits themselves, use the main window.
