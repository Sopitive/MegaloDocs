Object

Object variables refer to physical assets on the map, including vehicles, weapons, and Forge props. In MegaloEdit, object manipulation often involves spawning new items or adjusting their physical boundaries.

Actions

create_object
Reach Halo 4 H2A

Spawns a new object of the specified type at the location of the calling object.

Syntax: object.create_object(type, offset)

RVT Equivalent: object.place_at_me

delete_object
Reach Halo 4 H2A

Deletes the specified object from the game world.

RVT Equivalent: object.delete

set_boundary
Reach Halo 4 H2A

Sets the shape and size of the object's boundary (e.g., Sphere, Box, Cylinder).

RVT Equivalent: object.set_shape

set_invincibility
Reach Halo 4 H2A

Disables damage taken by the object.

RVT Equivalent: object.set_invincibility

set_hidden
Reach Halo 4 H2A

Makes the object invisible. Note that collision and interaction may still exist depending on the game engine.

RVT Equivalent: object.set_hidden

attach_to
Reach Halo 4 H2A

Parent the current object to another object, maintaining a relative offset.

RVT Equivalent: object.attach_to

set_scale
Reach Halo 4 H2A

Changes the visual and physical scale of the object. Supported scales vary by object type.

RVT Equivalent: object.set_scale

Member Conditions

object_is_type
Reach Halo 4 H2A

Returns true if the object belongs to the specified object type.

RVT Equivalent: object.is_of_type

has_forge_label
Reach

Checks if the object has a specific Forge label assigned to it.

RVT Equivalent: object.has_forge_label
