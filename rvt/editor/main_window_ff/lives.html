Player Lives

The player lives page allows you to control how many lives players on each team have.

The fields available to edit are as follows:

Spartan Starting Lives
Control the number of lives that Spartans start with. The value "unlimited" exists in place of -1 (i.e. to reach it, decrease the number of lives to "0 lives" and then decrease it again).
Max Spartan Extra Lives
Control the maximum number of extra lives that Spartans are allowed to have.
Reward for Killing Elite Players
Control the number of lives that Spartans earn for killing enemy players.
Elite Starting Lives
Control the number of lives that Elites start with. The value "unlimited" exists in place of -1 (i.e. to reach it, decrease the number of lives to "0 lives" and then decrease it again).
