object.set_waypoint_range

Change the distances at which waypoints are visible.

Arguments

min

A number between -1 and 100, inclusive. A value of 1 represents one Forge unit. The player will not be able to see the waypoint unless they are further away than this distance.

max

A number between -1 and 100, inclusive. A value of 1 represents one Forge unit. The player will not be able to see the waypoint unless they are closer than this distance.

Example


for each object with label "koth_hill" do
   current_object.set_waypoint_range(0, 50)
end

Notes

  • Most Megalo functions that work with distances use a value of 1 to represent one tenth of a Forge unit. However, this function uses a value of 1 to represent one entire Forge unit. Be mindful of the difference when using this function together with the others.
