object.kill

Kill an object.

Arguments

silent

Controls whether the object's death counts toward any statistics.

Example


for each object with label "avoid_me" do
   for each player do
      if current_object.shape_contains(current_player.biped) then
         current_player.biped.kill(false)
      end
   end
end
