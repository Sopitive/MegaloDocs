Game

The Game type (or ns_game) governs the global lifecycle of a match, including timers, round termination, and HUD messages.

Actions

end_round
Reach Halo 4 H2A

Ends the current round immediately. The game will proceed to the next round if applicable, or the post-game carnage report.

RVT Equivalent: game.end_round

end_game
Reach Halo 4 H2A

Ends the entire match immediately, regardless of current round status.

RVT Equivalent: game.end_game

hud_post_message
Reach Halo 4 H2A

Displays a temporary message to the specified player or team in their HUD feed.

timer_set_rate
Reach Halo 4 H2A

Adjusts the countdown or countup speed of a global timer.
