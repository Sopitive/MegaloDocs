API Reference

MegaloEdit scripting uses established object-oriented syntax to manage game world state. Select a type below to view its available actions and conditions.

Player

Score, traits, species, loadouts, and movement for match participants.

Object

Spawning, deletion, shapes, and collision management for map assets.

Team

Team scoring, member lists, and team-specific game options.

Game

Round termination, global HUD messages, and match lifecycle.
