MegaloEdit Actions

Actions are the primary way to manipulate game state, players, and objects within MegaloEdit triggers. Many of these actions have direct equivalents in the Reach Variant Tool (RVT) scripting language.

Player Actions

set_score

Sets or modifies the score for a specific player.

RVT Equivalent: player.score

apply_player_traits

Applies a set of scripted traits to the specified player for the duration of the current tick or a set interval.

RVT Equivalent: player.apply_traits

get_button_time

Returns the number of ticks a player has been holding a specific input button.

RVT Equivalent: player.get_button_time

Object Actions

create_object

Spawns a new object of a specific type at a target location or relative to another object.

RVT Equivalent: object.place_at_me

delete_object

Permanently removes an object from the map.

RVT Equivalent: object.delete

set_boundary

Sets the physical shape (sphere, box, etc.) for a specified object, often used for objective zones.

RVT Equivalent: object.set_shape

UI & Navpoint Actions

navpoint_set_visible

Sets the visibility of an objective navpoint on an object for specific players or teams.

RVT Equivalent: object.set_waypoint_visibility

hud_post_message

Displays a message to the kill feed or HUD message area.

Game Management

end_round

Ends the current round immediately.

RVT Equivalent: game.end_round

end_game

Ends the entire match immediately.

RVT Equivalent: game.end_game
