player.money

Type: Number

This property represents the player's current requisition money. Requisition money was part of a scrapped system during Halo: Reach's development, but this variable remains accessible to scripts for custom economy systems.

This value is read-only.

Notes

  • While the original engine logic for "purchasing" items with this money is mostly non-functional, you can still read and manually modify the value to implement your own currency system.
