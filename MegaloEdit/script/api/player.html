Player

The Player type refers to human or AI participants in a match. In MegaloEdit, the syntax for these actions is consistent across multiple games, though some actions are specific to certain titles.

Actions

set_score
Reach Halo 4 H2A

Sets the current score for the player to the specified value.

Syntax: player.set_score(number)

RVT Equivalent: player.score

add_weapon
Reach Halo 4 H2A

Gives an existing weapon object to the player.

Syntax: player.add_weapon(object)

RVT Equivalent: player.add_weapon

force_into_vehicle
Reach Halo 4 H2A

Forces the player into the specified vehicle. If no seat is specified, the player enters the first available seat.

Syntax: player.force_into_vehicle(object)

RVT Equivalent: player.force_into_vehicle

set_biped
Reach Halo 4 H2A

Changes the player's biped type (e.g., from Spartan to Elite or Monitor).

Syntax: player.set_biped(type)

RVT Equivalent: player.set_biped

set_objective_text
Reach Halo 4 H2A

Updates the objective text displayed on the player's HUD.

Syntax: player.set_objective_text(string)

RVT Equivalent: player.set_objective_text

apply_player_traits
Reach Halo 4 H2A

Displays a set of scripted traits to the specified player.

RVT Equivalent: player.apply_traits

get_button_time
Reach Halo 4 H2A

Returns the total number of ticks that a player has been holding an input button.

RVT Equivalent: player.get_button_time

set_loadout_palette
Reach Halo 4 H2A

Sets which loadout palette is currently available for the player to select from.

RVT Equivalent: player.set_loadout_palette

Properties & Conditions

player_is_spartan
Reach Halo 4 H2A

Returns true if the player's model species currently matches a Spartan.

RVT Equivalent: player.is_spartan

is_elite
Reach H2A

Returns true if the player's model species currently matches an Elite.

RVT Equivalent: player.is_elite

is_not_respawning
Reach Halo 4 H2A

Returns true if the player is currently alive and not in the process of respawning.

RVT Equivalent: player.is_not_respawning

is_out_of_bounds
Reach Halo 4 H2A

Returns true if the player is currently outside the soft kill or return-to-battle boundaries.

RVT Equivalent: player.is_out_of_bounds
