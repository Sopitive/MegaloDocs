local_team

a.k.a. hud_player_teamType: Team

This is the team to which the local player belongs. This value can be passed to HUD widgets as part of a format string to display information about the team.

This value is read-only.

Example

alias owned_vehicle_count = team.number[0]

for each player do
   script_widget[0].set_text("Your team has %n vehicles.", local_team.owned_vehicle_count)
end
