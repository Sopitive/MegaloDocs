Getting Started with MegaloEdit

MegaloEdit is a powerful, text-based tool for creating complex Megalo gametypes. Follow these steps to set up your environment correctly.

Installation & Steam Requirements

In order to use MegaloEdit, you must first own the Master Chief Collection (MCC) on Steam. Each game within the MCC has its own mod tools. Note that GamePass or Windows Store versions do not grant access to these tools.

  • HREK: Halo Reach mod tools.
  • H4EK: Halo 4 mod tools.
  • H2AMPEK: Halo 2: Anniversary mod tools.

Extracting the Tools

  1. Navigate to your Steam installation: Steam\steamapps\common.
  2. Open the desired -EK folder (e.g., HREK).
  3. Find and run (Extract).bat (e.g., HREK (Extract).bat). This will extract the compressed data into its final locations.

Setting up Directories

MegaloEdit requires specific folders to build and test gametypes successfully.

Megalodocs Folder
Within the extracted maps folder, create a new directory named megalo. This is where your compiled .mglo files will be generated. Without this folder, builds will fail.
Development Source
Official developer gametypes are usually located in data\multiplayer\megalo. Note that some older gametypes may use deprecated features and might not be fully compatible with current builds.

Hot Reloading

Hot reloading allow you to test your changes in custom games almost instantly.

  1. Navigate to %AppData%\LocalLow\MCC\Temporary.
  2. Open the corresponding game folder (e.g., Halo Reach).
  3. Create a folder named HotReload.
  4. Move your compiled .mglo files into this folder to use them in the in-game custom game menu.
