Interface & Scoring

Learn how to add elements to the player's HUD and configure how scores are tracked and displayed.

HUD Widgets

HUD widgets add text, icons, and meters to specific locations on a player's screen. You can have up to four widgets per gametype. Note that while Reach supports many locations, H4 & H2 display all widgets in the low_center location.

hud_widgets
    widget_name_1 low_center
    widget_name_2 top_left
    widget_name_3 high_right
    widget_name_4 bottom_center
end

Game Stats

Game stats are used to track and display player performance. Note that the MCC does not support displaying custom stats in the post-game after-action report, but they still influence player position on the scoreboard.

game_stats
    stat_name1 number "Kills" none 1 ; positive sorting
    stat_name2 percentage "Accuracy" none 0
end

Player Ratings

Player ratings are used to calculate a player's performance relative to others. This was extensively used for the original Halo: Reach Arena playlists.

player_rating
    show_in_scoreboard true
    kill_weight 1
    assist_weight 0.5
    betrayal_weight 1
    death_weight 0.5
    base_value 1000
    range 1000
end
